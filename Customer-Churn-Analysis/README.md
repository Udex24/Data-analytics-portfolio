# Customer Churn Analysis

## Problem
Telecoms Subscription Ltd experienced high customer churn and needed
to understand key drivers of customer attrition.

## Dataset
- Rows: 64,375
- Key features: Tenure, Payment Delay, Support Calls, Churn

## Tools Used
- Excel

## Key Insights
- 47% overall churn rate
- Payment delays accounted for ~92% of churn
- Long-tenured customers churned more than new customers

## Recommendations
- Flexible payment plans
- Targeted retention programs
